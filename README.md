# RealtimeMonitor
Realtime Server Monitor

## Setup

### Requirements

```
npm node 
vnstat
```

### Setup

```
npm install
bower install
``` 

## Setings
```
cp settings_example.json settings.json
```
* **port**: start webserver in this port
* **ifaces**: filter the given interfaces.

## Run

```
node app.js
```
oder
```
nodemon app.js
```
